Below are examples of different lists in HTML

This is an example of an ordered list. Notice how the items are numerically listed:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

This can also be done in alphebetical order:

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4

This is an example of an unordered list. Notice how the items are only listed by bullet points instead of numbers:

  • Item 1
  • Item 2
  • Item 3
  • Item 4

      • Here is an example of just a plain, description list:

      Item 1
      Item 2
      Item 3
      Item 4
